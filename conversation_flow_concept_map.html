Business Logic & Rules

üïê 24-Hour Cancellation Policy

If student cancels within 24 hours of appointment:

  • Warn about cancellation fee
  • Require explicit confirmation
  • Set Charge="Y" in cancelAppointment API
  • Log reason and timestamp
üìç Location Mapping

Internal codes vs student-facing names:

  • Brooklyn = "BK" ‚Üí 2444 Linden Blvd
  • Queens = "AS" ‚Üí 3528 19th Ave
  • Always show full addresses to students
  • Use codes only for API calls
üéØ Lesson Duration Logic

Duration options by lesson type:

  • Behind-the-wheel: 1.5, 2, 3 hours
  • Road test prep: 2, 2.25, 3 hours
  • Highway practice: 2, 3 hours
  • Pass duration to API without calculation
üë®‚Äçüè´ Instructor Assignment

Instructor selection priority:

  • Ask if student wants previous instructor
  • Check instructor availability first
  • Show all available slots from API
  • Let student choose instructor + time
üîê Authentication Flow

Multi-step verification process:

  • Always ask for driver's license number first
  • Call getStudentInfo immediately
  • If not found, ask for DOB as backup
  • Store license number in session memory
üí¨ Conversation Memory

Context preservation rules:

  • Remember license number for entire session
  • Cache student info for 15 minutes
  • Track current conversation state
  • Clear sensitive data after 30 minutes
